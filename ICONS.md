# Icons Setup

The PWA manifest references icon files that need to be created. You can:

1. **Generate icons** using a tool like:
   - https://www.pwabuilder.com/imageGenerator
   - https://realfavicongenerator.net/
   - https://www.favicon-generator.org/

2. **Create the icons directory**:
   ```
   mkdir icons
   ```

3. **Required icon sizes** (as per manifest.json):
   - 72x72
   - 96x96
   - 128x128
   - 144x144
   - 152x152
   - 192x192
   - 384x384
   - 512x512

4. **Place icons** in the `icons/` directory with the naming convention:
   - `icon-72x72.png`
   - `icon-96x96.png`
   - etc.

For quick testing, you can use placeholder images or emoji-based icons.
